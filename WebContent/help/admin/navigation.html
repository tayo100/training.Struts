Navigating Administrative console

Once you are in the scope of Administrative console (/admin/ pages) the top-level menu lists most important tools for Administrators. The Public site link will bring you back to the application and leave Administrative console. Clicking the logo within the console will open up Administrator's Dashboard.

Dashboard

Dashboard gives you access to other Administrative tasks:

  • Scheduled jobs
  • Lucene reindex
  • Block access to application
  • JRE status

User impersonation

In the top right corner you will find the Fast user switch applet. Read more about switching to other users.
