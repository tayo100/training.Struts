Project profile

The project profile page displays most relevant information about the project. More detail can be found by navigating to various tabs (e.g. Agreements, Financials, etc).

Several attributes are core to every project:

  • Project number: a sequential PJ number generated by the system when project is created.
  • Status: Project status, one of Not started, Withdrawn, Active, In closeout, Closed.
  • Project title
  • Description: brief description of a project. See "Narratives" for more.
  • Start date: Actual start date of project, based on Agreement(s).
  • End date: End date of project, based on Agreement(s).
  • Extension date: With every agreed extension, this value determines actual project end date.

Cost centers

Project will usually have one or more cost centers. This section lists all cost centers associated with the project. Clicking All costcenters of the project will open up a screen summarizing all expenditures and budgets on all project's cost centers.
