Tag builder

Tag builder allows CRM users to re-use existing sets of tags. Builder will inspect the database for different tag categories and render all category sections with all their tags on one screen. By ticking/checking the box next to the tag, that assigns the tag to the entity.

Tag builder in action

After selecting the tags you wish to apply or deselecting the ones already applied, click the Save tags button at the bottom of the page. All affected tags will be updated.

Percentage

In CRM, some tags may carry a percentage value. When using percentage values, make sure they add up to 100%!
